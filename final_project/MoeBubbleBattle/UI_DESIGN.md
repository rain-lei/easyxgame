# 《萌泡大作战》完整 UI 设计规范（单人版）

## 1. 设计目标

本规范用于固定《萌泡大作战》的视觉语言、界面布局、交互反馈和场景切换。游戏采用固定 `960 × 720` 窗口，面向单人键盘游玩，同时为所有可见菜单按钮提供鼠标悬停和左键点击。界面应当可爱、清楚、易演示。

视觉关键词：Q版、手绘素描、浅色水彩、圆角卡片、透明水泡、轻松明亮。

## 2. 视觉规范

### 2.1 色彩

| 名称 | 色值 | 用途 |
| --- | --- | --- |
| 奶油纸白 | `#FFF8ED` | 全局背景、卡片底色 |
| 深灰蓝 | `#344C5A` | 正文、边框、主要轮廓 |
| 水泡蓝 | `#66D5E8` | 主按钮、玩家、水泡、选中状态 |
| 深水蓝 | `#289BB0` | 主按钮悬停/按下、标题阴影 |
| 珊瑚粉 | `#FF8F82` | 重点提示、兔子角色、失败提示 |
| 薄荷绿 | `#91D8BF` | 成功提示、护盾和安全状态 |
| 蜂蜜黄 | `#F5C768` | 分数、奖励、关卡完成 |
| 危险红 | `#E75D72` | 生命减少、危险区、失败按钮 |
| 浅阴影 | `#DCCDBD` | 卡片阴影、层级分隔 |

### 2.2 字体

- 中文：微软雅黑，回退为黑体。
- 标题：48–64 px，加粗，深灰蓝描边与浅蓝阴影。
- 一级界面标题：36–42 px。
- 按钮：24–28 px。
- HUD 主数字：24 px。
- 说明文字：18–20 px。
- 所有文字与背景保证明显明暗对比，不在复杂水泡纹理上直接放字。

### 2.3 形状与线条

- 大卡片圆角：24 px；普通按钮圆角：18 px；小徽章圆角：12 px。
- 主轮廓 3 px 深灰蓝，内部装饰线 1–2 px。
- 卡片向右下偏移 6 px 绘制浅阴影。
- 水泡由三层圆形组成：浅蓝主体、白色高光、小型外环。
- 素描感只用于装饰短线和角色轮廓，不影响按钮边界和文字清晰度。

### 2.4 间距

- 基础间距单位：8 px。
- 界面安全边距：24 px。
- 卡片内边距：24–32 px。
- 同组按钮间距：16 px。
- 点击区域不得小于 `160 × 52` px。

## 3. 角色与图标

角色设定参考 `assets/concepts/chibi_character_concepts.png`。四个角色分别采用全能、疾速、爆破、守护定位，初始属性不同但共享相同强化上限。

地图内角色使用 `assets/sprites/player_walk_sheet.png`。精灵表按四行三列排列：四行对应熊、兔、猫、狗，三列对应左脚、站立、右脚。程序以六阶段逻辑周期在跨步之间插入回中帧，跨步帧缩小 4 px 并进行 1 px 重心校正。角色可见宽度约等于一格 `44 px` 步道；帽子和背包只参与显示，实际碰撞使用约 `20 × 20 px` 的脚底区域。

| 角色 | 主色 | 代号 | UI特点 |
| --- | --- | --- | --- |
| 蓝熊少年 | 水泡蓝 | `Bear` | 默认角色、圆形护目镜、背包水罐 |
| 珊瑚兔少女 | 珊瑚粉 | `Rabbit` | 长耳帽、星形发卡、小水泡发射器 |
| 薄荷猫发明家 | 薄荷绿 | `Cat` | 圆眼镜、工具袋、水泡仪表 |
| 蜂蜜犬侦察员 | 蜂蜜黄 | `Dog` | 飞扬围巾、护膝、小水壶 |

HUD 和地图拾取物使用 `assets/ui/item_icons_v1.png` 四格透明水彩图集：

- 生命仍使用粉色心形。
- 水泡容量：蓝色水泡图标。
- 水浪范围：四向箭头图标。
- 速度：黄色小靴子。
- 护盾：薄荷绿色圆盾。
- 敌人：紫色小怪头像。

## 4. 界面清单

### UI-M01 主菜单

布局：

- 左上角 `24, 20`：版本小标签“单人闯关”。
- 中上方：游戏标题“萌泡大作战”，标题周围漂浮 5–7 个小水泡。
- 左侧 `100, 210`：当前选择角色的大幅 Q 版立绘。
- 右侧 `560, 240`：纵向主按钮组。
- 底部：`Enter 确认  ↑↓ 选择  Esc 退出`。

按钮顺序：

1. `开始游戏`
2. `角色选择`
3. `操作说明`
4. `退出游戏`

状态：默认按钮奶油白；选中按钮水泡蓝并放大约 4%；按下时颜色变深并向下偏移 2 px。

### UI-C01 角色选择

- 顶部标题：“选择你的泡泡搭档”。
- 中部四张角色卡横向排列，每张约 `196 × 420` px。
- 卡片包含角色全身、中文昵称、职业、生命/移速文字和容量/范围/护盾徽章。
- 当前卡片使用对应角色主色边框和水泡光环。
- 底部按钮：`确认选择`、`返回主菜单`。
- 键盘：左右键切换，Enter 确认，Esc 返回；鼠标悬停卡片并点击确认按钮也可操作。

### UI-I01 操作说明

- 左侧：键盘按键示意，突出 `W/A/S/D`、方向键、`Space`、`P`。
- 右侧：四条规则卡片。
  1. 移动并寻找安全路线。
  2. 放置水泡，2.5 秒后释放十字水浪。
  3. 水浪可破坏木箱并连锁触发水泡。
  4. 淘汰全部敌人即可过关。
- 底部列出四种道具图标和效果。
- `Enter/Esc` 返回。

### UI-G01 游戏主界面

固定布局：

- 顶部状态条：`x=24, y=16, w=660, h=48`。
- 地图：`x=24, y=76, w=660, h=572`，15列×13行，每格44 px。
- 右侧HUD：`x=708, y=24, w=228, h=672`。
- 底部地图提示条：`x=24, y=660, w=660, h=40`。

顶部状态条从左到右：

- 关卡：`第 1 / 3 关`
- 剩余敌人：紫色头像 + 数字；Boss 存活时切换为 `首领 HP 当前值 / 最大值`
- 当前分数：黄色数字

右侧HUD从上到下：

1. 圆形角色头像与角色昵称。
2. 生命心形，最多显示 4 个。
3. 强化面板：容量、范围、速度、护盾四张 `42 × 88` 等宽状态卡横向排列。每张卡从上到下依次放置透明图标、中文名称和独立数值胶囊；三位速度值缩小字号但不改变卡片尺寸。
4. 关卡目标：“淘汰全部敌人”。
5. 小型操作提示：移动、放泡、暂停。
6. 当前状态提示，例如“护盾已激活”。

地图视觉：

- 草地/地砖采用奶油色与淡薄荷色交替。
- 固定墙使用蓝灰色圆角石块。
- 木箱使用蜂蜜黄和棕色十字木纹。
- 玩家脚下有主色圆形投影；敌人脚下为紫色投影。
- 危险水泡最后 0.7 秒快速闪烁，水浪格有明显浅蓝十字方向。

### UI-P01 暂停界面

- 保留游戏画面，但覆盖深灰蓝网点半透明效果。
- 中央圆角卡片 `420 × 360` px。
- 标题：“游戏暂停”。
- 按钮：`继续游戏`、`重新开始本关`、`操作说明`、`返回主菜单`。
- 使用 P/Enter 继续，Esc 返回游戏，不直接退出程序。

### UI-L01 关卡完成

- 中央金黄色水泡奖章与标题“关卡完成！”。
- 显示本关得分、破坏木箱数、获得道具数、剩余生命。
- 三颗星根据表现点亮，但星级只作展示，不影响过关。
- 按钮：`进入下一关`，第三关完成后改为 `查看通关结果`。

### UI-F01 游戏失败

- 珊瑚粉色卡片，角色坐在破掉的水泡旁，不使用恐怖或沉重画面。
- 标题：“挑战失败”。
- 显示最终分数与到达关卡。
- 按钮：`重新挑战`、`返回主菜单`。

### UI-V01 最终通关

- 四个角色在水泡和彩纸中庆祝，当前角色位于最前方。
- 标题：“全部通关！”。
- 显示最终分数、总用时、木箱破坏数、道具收集数。
- 按钮：`再玩一次`、`返回主菜单`。

### UI-D01 退出确认

- 小型确认框：“确定要退出游戏吗？”
- 按钮：`继续玩`、`退出游戏`。
- 默认选中“继续玩”，防止误按 Esc 直接关闭。

## 5. 场景流程

```text
启动
  ↓
主菜单 ─→ 角色选择 ─┐
  ├────→ 操作说明 ──┤
  ├────→ 退出确认   │
  └────→ 游戏中 ←──┘
            ↕
          暂停
            ↓
    关卡完成 → 下一关 → 最终通关
            ↘
            游戏失败
```

## 6. 动效与反馈

- 按钮切换：120 ms 缩放与颜色变化。
- 场景切换：200 ms 水泡扩散遮罩；EasyX 实现时可用逐渐扩大的同心圆模拟。
- 得分：黄色 `+20` / `+300` 向上飘动约 600 ms。
- 受伤：角色闪烁 1.5 秒，HUD 心形轻微抖动。
- 获得道具：对应图标从地图飞向右侧强化面板。
- 水泡倒计时：由缓慢呼吸逐步加快，最后 0.7 秒切换危险红轮廓。
- 过关：暂停游戏对象，播放星星与水泡粒子，再出现结算卡片。

## 7. 单人模式约束

- 游戏只维护一个本地玩家实例。
- 角色选择决定外观、主题色和一套差异化初始属性。
- 对手全部由 AI 控制，第一关为巡逻型，后续加入追踪型。
- 不设计第二套按键、双人分屏或玩家对战结算。
- HUD 只展示一个头像与一组能力状态。

## 8. EasyX 实现映射

| UI元素 | EasyX实现 |
| --- | --- |
| 圆角卡片/按钮 | `fillroundrect`、`roundrect` |
| 水泡 | `solidcircle`、`circle`、白色高光圆 |
| Q版角色 | 圆、椭圆、多边形和短线组合 |
| 阴影 | 先绘制偏移浅色形状，再绘制主体 |
| 图标 | 透明 PNG 图集裁切；加载失败时回退为几何图形和字符 |
| 文本 | `settextstyle`、`outtextxy`、`textwidth` |
| 选中动画 | 根据时间改变尺寸、位置和颜色 |
| 场景切换 | 扩大同心圆或短暂纯色遮罩 |

最终代码中的坐标、颜色和字体尺寸以本文件为准。生成式 UI 视觉稿用于确定风格与构图，实际文字和交互由 EasyX 程序精确绘制。
